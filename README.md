# Executive Recruitment & Hiring Analytics Dashboard

## 📊 Project Overview
This interactive Power BI dashboard transforms a raw dataset of 10,000 candidate records into executive-level hiring insights. Developed during my training project with Ededge, this end-to-end business intelligence solution optimizes recruitment pipelines, diagnoses processing delays, and maps historical offer acceptance behaviors.

## 💡 Key Business Insights Discovered
* **Funnel Bottlenecks (Task 2):** While 7.11K candidates successfully clear initial screening, the steepest operational drop-off occurs between the HR Interview phase (4.04K) and formal Offer Generation (2.95K).
* **Operational Delays (Task 3):** Data Scientist roles consistently experience the longest processing timeline (~32.4 days), signaling a clear need for technical evaluation optimization.
* **Conversion Hurdles (Task 5):** The scatter plot reveals that Business Analyst hires drop to the absolute lowest Offer Acceptance Rate on the grid, representing a critical focus area for candidate engagement. 
* **Pipeline Efficiency (Task 5):** Software Engineer streams represent the most efficient sourcing pipeline, capturing the fastest overall average time-to-hire (~31.6 days).

## 🛠️ Technical Process & Skills Used
* **Data Transformation:** Engineered data clean-up procedures in Power Query Editor, including unpivoting complex tracking metrics to establish a vertical, scalable multi-stage data architecture.
* **DAX Calculation Modeling:** Formulated optimized DAX measures utilizing conditional calculation blocks (`DIVIDE`, `CALCULATE`, `COUNTA`) to dynamically map monthly metrics.
* **Advanced Data Visualization:** Constructed comprehensive custom canvas landscapes matching corporate theme presets. Implemented highly correlated Combo charts, Matrix tables, and multi-axis Scatter plots.
